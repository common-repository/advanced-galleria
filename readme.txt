== Description ==

Replaces the Wordpress default "[gallery]" shortcode to insert a Galleria Slideshow.

== Installation ==

1. Upload `advanced-galleria` and all its contents to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use the `[gallery]` shorttag in your Pages and Posts to insert a Galleria slideshow containing all the images attached to that Page or Post
